
# 🏓 Pickleball Mixed Doubles Round Robin

A free, no-install web app for organizing pickleball round robin sessions. Just open the file in any browser — no account, no app, no internet connection required after the first load.

---

## ✨ Features

- **Player signup** — Register up to 12 players and 2 substitutes
- **Auto-generated schedule** — 8 rounds across 3 courts, fully balanced rotation
- **Court number selector** — Choose any court numbers 1–20 for your session
- **Game start roles** — Server, server's partner, receiver, and receiver's partner assigned each round
- **Score tracking** — Enter scores round by round with live winner preview
- **Standings** — Auto-calculated leaderboard sorted by wins, win %, and point differential
- **Self-removal** — Players can remove themselves by name without an organizer
- **Venue & time tab** — Log your location, date, start/end time, and round duration with an auto-calculated round timeline
- **Print schedule** — Clean print layout for posting at the courts
- **Works offline** — Everything runs in the browser, no server needed

---

## 📋 Quick Start Guide

1. **Set court numbers** — Use the dropdowns at the top to pick which court numbers (1–20) you're playing on
2. **Add players** — Type each player's name and click **Add Player** (up to 12)
3. **Add substitutes** — Optionally add up to 2 subs who rotate in when players sit out
4. **Generate schedule** — Click the **Generate Schedule** button
5. **Play!** — Share the screen or print the schedule for players to follow
6. **Enter scores** — Go to the **Scores** tab after each round
7. **Check standings** — View the live leaderboard in the **Standings** tab
